## Installation / Instalação:
- Clone the repository / Clone o repositório:
````
  git clone https://github.com/Armandoedu/JogoPlataforma.git
````

- Run the file / Rode o arquivo:
````
  python "/JogoPlataforma/Jogo_Rua_Dispensario/Main.py"
````
